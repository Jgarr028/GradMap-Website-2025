User Stories

As a Student, I need to be able to

  • Enter my degree program, and plan so that Gradmap can generate personalized schedules.
  • Choose my preferred time blocks, unavailable times, etc.
  • Automatically generate a conflict-free schedule.
  • View alternatives for each semester or course.
  • See indicators for required or critical courses.
  • Share my schedule with an advisor for review.
  • Receive a notification if my plan conflicts.

As an Academic Advisor, I need to be able to

  • Review student schedules.
  • Approve, reject, or suggest modifications.
  • Receive alerts for students whose schedules contain conflicts or missing requisites.
  • Suggest alternative courses or plans for students.
  • Access past student schedules to track progress.

As an Admin, I need to be able to

  • Manage student and advisor accounts.
  • Customize system settings.
  • Monitor system activity. For example, schedule logs, conflicts, and alerts.
  • Ensure data privacy and compliance for account info.
